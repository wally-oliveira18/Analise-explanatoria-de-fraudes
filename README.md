# Análise explanatória de fraudes de cartões 

Este projeto apresenta uma Análise Exploratória de Dados (EDA) utilizando MySQL em um conjunto de transações de cartões de crédito.

O objetivo é explorar os dados por meio de consultas SQL básicas, identificando padrões de comportamento e fatores relacionados às fraudes.

Todas as consultas foram desenvolvidas utilizando apenas comandos fundamentais da linguagem SQL.

## Objetivos

Este projeto tem como objetivo explorar a distribuição das transações e analisar o comportamento de fraudes, a fim de determinar os comportamentos de riscos em transações nacionais e internanais, bem como os padõres relacionados aos dispositivos e canais de compra em fraudes de cartões. 

- ## Tecnologias

Para o desenvolvimento do projeto foi utlizado a ferramenta MySQL Workbench. 


- ## Comandos utilizados

Para as análises foram utilizadas os seguintes comando em SQL: 
    - SELECT
    - WHERE
    - GROUP BY
    - ORDER BY
    - HAVING
    - COUNT()
    - AVG()
    - SUM()
    - MAX()
    - MIN()
    - DISTINCT
    - AND
    - OR
    - IN
